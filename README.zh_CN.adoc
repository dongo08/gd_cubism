== 关于 GDCubism

本软件（以下简称 GDCubism）是一个**非官方**的 GDExtension，使您可以通过 GDScript 或 C# 使用 link:https://www.live2d.com/download/cubism-sdk/[Live2D Cubism SDK]。

[NOTE]
====
此 GDExtension 适用于 Godot Engine 4.3 及以上版本。（如果需要在 4.1 或 4.2 中使用，请使用 link:https://github.com/MizunagiKB/gd_cubism/releases/tag/v0.8.2-godot4.1[v0.8.2-godot4.1]）
====

[IMPORTANT]
====
从 0.9 版本开始，渲染方式改为直接渲染，大幅降低了 Godot Engine 的处理负载和 GPU 内存消耗。

同时也有一些变更。最大的变化是 _GDCubismUserModel_ 不再是 _SubViewport_。

如果希望像以前一样使用，请将 _GDCubismUserModel_ 放在 _SubViewport_ 下使用。（关于此变更的演示已添加 `demo_fade.tscn`）
====

GDCubism 内部使用了 Live2D 公司公开的 link:https://github.com/Live2D/CubismNativeFramework[Cubism Native Framework]。

请不要就 GDCubism 的相关问题联系开发方 Live2D 公司。
